##### Visualiseur de données climatiques

Commencez en choisissant un des ensembles de données disponibles. Une description détaillée de l'ensemble de données choisi est affichée sous la carte. Vous pouvez faire un zoom sur un endroit précis ou cliquer sur les données pour obtenir plus d'information.

Pour obtenir de l’aide, cliquez sur l’icône « ? » dans le coin inférieur droit de la carte.
